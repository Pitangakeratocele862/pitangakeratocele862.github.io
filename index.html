🔍 What is this tool?

Knowledge Gravity Lab helps you organize your notes. If you store your thoughts in Markdown files or use Obsidian, your collection can become messy over time. This tool scans your files and identifies notes that need your attention. It works entirely on your computer. Your data never leaves your machine. It does not use external services or artificial intelligence. It uses logical rules to find patterns in your vault.

📋 Key Features

  • • Center Topics: Find the notes that anchor your knowledge base.
  • • Orphan Detection: Spot notes that lack links to other ideas.
  • • Noise Reduction: Identify weak notes that clutter your system.
  • • Action Items: Get suggestions on how to improve individual files.

🚀 Getting Started

Follow these steps to set up the application on your Windows computer.

  1. Requirements: Ensure you run Windows 10 or 11. You need at least 200MB of free disk space.
  2. Download: Visit this page to download the latest Windows installer.
  3. Installation: Locate the file in your Downloads folder. Double-click the file to start the installation wizard. Follow the prompts on the screen.
  4. Usage: Open the application from your Start Menu after installation finishes.

🖥️ Using the Application

Once you open the software, it asks you to select a folder. Point the application to the folder that contains your Markdown notes. The tool scans your files. This process takes a few moments depending on the size of your vault. Once the scan finishes, the dashboard displays a summary of your note health.

You see a list of categories on the left side of the screen. Click on "Orphans" to view notes that do not connect to the rest of your knowledge. Click on "Noise" to see files the software identifies as low-value. Each file shows a clear action recommendation to help you fix the issue.

⚙️ Privacy and Security

This tool runs entirely offline. It does not contain trackers. It does not connect to the internet to verify your notes. You own your data. When you delete a note using this tool, the software simply performs a standard file deletion on your drive. Always keep a backup of your important documents before you perform large cleanups.

💡 Frequently Asked Questions

Does this tool modify my original files? It reads your files to perform analysis. It only changes files when you explicitly request a correction or deletion within the user interface.

Can I use this with other apps? Yes. This tool recognizes standard Markdown formatting compatible with Obsidian, Typora, and other text editors.

Does it use LLMs? No. The software uses deterministic rules. It relies on link counts, word counts, and metadata to provide its results. This ensures your results remain consistent every time you run a scan.
